# 💰 Currency Bot — Курсы валют ЦБ РФ

Telegram-бот для получения актуальных курсов валют с сайта Центрального Банка России.

### Возможности
- Реaltime курсы валют (обновляются при запросе)
- Команды для популярных валют: `/usd`, `/eur`, `/cny`, `/gbp`, `/try`
- Команда `/all` — показывает первые 15 валют
- Кэширование данных (не нагружает сайт при каждом запросе)
- Чистый и понятный вывод

### Технологии
- **Python 3**
- **aiogram 3** — асинхронный фреймворк для Telegram-ботов
- **BeautifulSoup4** + **requests** — парсинг данных
- **JSON** — сохранение данных

### Как запустить локально

```bash
# 1. Клонируй репозиторий
git clone https://github.com/TimPalladin/Course-Bot.git
cd Course-Bot

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Создай файл config.py и вставь свой токен:
# TOKEN = "123456:ABC-DEF..."

# 4. Запусти бота
python main.py
